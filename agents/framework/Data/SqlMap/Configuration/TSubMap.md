`Prado\Data\SqlMap\Configuration\TSubMap`

A discriminator sub-map entry mapping a value to a result map.

## Description

`TSubMap` represents a single entry in a `TDiscriminator` mapping. It maps a column value to a result map name.

## Key Properties

| Property | Type | Description |
|----------|------|-------------|
| `Value` | `string` | The discriminator column value |
| `ResultMap` | `string` | The result map name to use |
