`Prado\Data\SqlMap\Statements\TSimpleDynamicSql`

Dynamic SQL with simple property substitution.

## Description

`TSimpleDynamicSql` handles dynamic SQL with `$property$` simple string substitution.

## Warning

`$property$` substitution is not safe with user input. Use `#property#` parameterized binding instead.
