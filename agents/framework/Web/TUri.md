## Overview

TUri represents a URI and decomposes it into its constituent parts: scheme, host, port, user, password, path, query, and fragment. It provides a structured way to parse and access URI components.

## Key Properties/Methods

- `getUri()` - Returns the full URI string
- `getScheme()` - Returns the scheme (e.g., 'http', 'https', 'ftp')
- `getHost()` - Returns the hostname
- `getPort()` - Returns the port number
- `getUser()` - Returns the username
- `getPassword()` - Returns the password
- `getPath()` - Returns the path component
- `getQuery()` - Returns the query string
- `getFragment()` - Returns the fragment (anchor)
