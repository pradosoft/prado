## Overview

Progress bar widget based on jQuery UI Progressbar. Extends [TActivePanel](../ActiveControls/TActivePanel.md).

## Key Properties/Methods

- `getOptions()` - Progressbar options (disabled, max, value)
- `getValidOptions()` - Valid options: disabled, max, value
- `getValidEvents()` - Events: change, complete, create
- `onChange($params)` - Raises OnChange event
- `onComplete($params)` - Raises OnComplete event
- `onCreate($params)` - Raises OnCreate event
