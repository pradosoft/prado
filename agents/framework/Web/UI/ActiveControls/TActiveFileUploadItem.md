# TActiveFileUploadItem

**Location:** `framework/Web/UI/ActiveControls/TActiveFileUploadItem.php`
**Namespace:** `Prado\Web\UI\ActiveControls`

## Overview

Represents a single uploaded file from TActiveFileUpload. Needed especially when Multiple file upload is enabled. Provides saveAs method to save the uploaded file to a permanent location.

## Key Properties/Methods

- `saveAs($fileName, $deleteTempFile)` - Saves uploaded file to specified location
