## Overview
Active counterpart to TButtonColumn for use in [TActiveDataGrid](./TActiveDataGrid.md). Creates [TActiveLinkButton](./TActiveLinkButton.md), [TActiveButton](./TActiveButton.md), or [TActiveImageButton](./TActiveImageButton.md) based on button type for each row.

## Key Properties/Methods

- `initializeCell($cell, $columnIndex, $itemType)` - Creates appropriate active button for item cells
