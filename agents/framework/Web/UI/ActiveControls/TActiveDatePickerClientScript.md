## Overview
Client-side options for [TActiveDatePicker](./TActiveDatePicker.md). Extends [TCallbackClientSide](./TCallbackClientSide.md) with `OnDateChanged` event handler that fires when the date picker's date changes, receiving the formatted date as parameter.

## Key Properties/Methods

- `setOnDateChanged($javascript)` / `getOnDateChanged()` - JavaScript to execute when date changes
