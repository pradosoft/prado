## Overview
Active counterpart to TImage. AlternateText, ImageUrl, and DescriptionUrl properties can be updated during callback response.

## Key Properties/Methods

- `setAlternateText($value)` - Sets alt text with client-side update
- `setImageUrl($value)` - Sets image URL with client-side update
- `setDescriptionUrl($value)` - Sets long description URL with client-side update
- `getActiveControl()` - Returns [TBaseActiveControl](./TBaseActiveControl.md) options
