## Overview
Active counterpart to TTemplateColumn for use in [TActiveDataGrid](./TActiveDataGrid.md). If sorting is allowed, replaces header links/buttons with active controls ([TActiveLinkButton](./TActiveLinkButton.md), [TActiveImageButton](./TActiveImageButton.md)) instead of regular postback controls.
