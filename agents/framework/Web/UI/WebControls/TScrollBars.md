**Namespace:** `Prado\Web\UI\WebControls`

## Overview

TScrollBars defines the enumerable type for scroll bar modes that can be applied to TPanel controls. It specifies when and how scroll bars appear in a panel.

## Values

- `None` - No scroll bars
- `Auto` - Scroll bars automatically appear when needed
- `Both` - Always show both horizontal and vertical scroll bars
- `Horizontal` - Show horizontal scroll bar only
- `Vertical` - Show vertical scroll bar only
