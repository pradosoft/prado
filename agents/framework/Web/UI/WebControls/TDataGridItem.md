## Overview
TDataGridItem represents an item in a TDataGrid control (heading, footer, or data item). Tracks item index, data source index, and item type.

## Key Properties/Methods

- `ItemIndex` - Zero-based index in Items collection
- `DataSourceIndex` - Index from bound data source
- `ItemType` - Type of item (Header, Footer, Item, etc.)
- `Data` - Associated data item
- `bubbleEvent()` - Bubbles OnCommand events to parent
