## Overview

TPagerMode defines the enumerable type for the possible modes that a TPager control can take.

## Key Properties/Methods

- `NextPrev` - Pager buttons displayed as next and previous pages
- `Numeric` - Pager buttons displayed as numeric page numbers
- `DropDownList` - Dropdown list used to select pages
