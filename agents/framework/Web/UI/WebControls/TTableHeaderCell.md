**Namespace:** `Prado\Web\UI\WebControls`

## Overview

TTableHeaderCell displays a table header cell (th element) on a Web page. It extends TTableCell and adds scope attribute for accessibility.

## Key Properties/Methods

- `getScope()` / `setScope($value)` - Scope of header cell (Row, Col)
- `getAbbreviatedText()` / `setAbbreviatedText($value)` - Abbreviation for the header
- `getCategoryText()` / `setCategoryText($value)` - Axis attribute (deprecated)
