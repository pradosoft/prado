## Overview
IListControlAdapter defines the interface for list control adapters that manage client-side selection behavior.

## Key Properties/Methods

- `setSelectedIndex($index)` - Select item by zero-based index
- `setSelectedIndices($indices)` - Select multiple items by indices
- `setSelectedValue($value)` - Select by item value
- `setSelectedValues($values)` - Select multiple by values
- `clearSelection()` - Clear all selections
