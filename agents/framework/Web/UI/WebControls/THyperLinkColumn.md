## Overview
THyperLinkColumn contains hyperlinks for each item in a DataGrid column. Text and URL can be static or bound to data fields.

## Key Properties/Methods

- `Text` - Static link caption
- `NavigateUrl` - Static navigation URL
- `DataTextField` / `DataNavigateUrlField` - Data fields for dynamic values
- `DataTextFormatString` / `DataNavigateUrlFormatString` - Format strings
- `ImageUrl` / `ImageWidth` / `ImageHeight` - Image properties
- `Target` - Target window for links
- `initializeCell()` - Creates hyperlink in cell
- `dataBindColumn()` - Binds data to link properties
