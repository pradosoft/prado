**Namespace:** `Prado\Web\UI\WebControls`

## Overview

THeader4 is a simple control for rendering h4 elements with theme support. It extends THtmlElement and allows themes to redefine the TagName via TWebControlDecorator.

## Key Properties/Methods

- `getDefaultTagName()` - Returns 'h4'
