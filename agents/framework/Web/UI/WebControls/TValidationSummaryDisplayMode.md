## Overview

TValidationSummaryDisplayMode defines the enumerable type for the possible modes that a TValidationSummary can organize and display collected error messages.

## Key Properties/Methods

- `SimpleList` - Error messages as plain list
- `SingleParagraph` - Messages concatenated into a paragraph
- `BulletList` - Messages as bulleted list
- `HeaderOnly` - Only header text displayed
