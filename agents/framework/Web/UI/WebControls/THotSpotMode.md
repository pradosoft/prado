## Overview

THotSpotMode defines behaviors when a hotspot is clicked.

## Constants

- `NotSet` - Not specified (defaults to Navigate)
- `Navigate` - Redirect to specified URL
- `PostBack` - Cause a postback
- `Inactive` - Hotspot is not clickable
