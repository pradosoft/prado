## Overview

TDataSize displays file sizes in human-readable format (bytes, KB, MB, GB, etc.) instead of raw byte counts.

## Key Properties/Methods

- `Size` - Size in bytes to display
- `UseMarketingSize` - Use base 1000 (marketing) vs 1024 (technical)
- `Abbreviate` - Use abbreviated units (B, KB, MB) vs full names
- `Culture` - Localization culture for formatting
