## Overview
TRangeValidationDataType defines the enumerable type for the possible data types that a range validator can validate upon. It extends TValidationDataType with an additional StringLength value.

## Key Properties/Methods

- `StringLength` - Check for string length
- Inherits from TValidationDataType: `Integer`, `Float`, `Date`, `String`
