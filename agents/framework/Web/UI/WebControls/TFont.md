## Overview

TFont encapsulates CSS style fields related to font settings. It provides properties for font name, size, and style attributes like bold, italic, underline, overline, and strikeout.

## Key Properties/Methods

- `getBold()` / `setBold()` - Gets or sets whether font is bold
- `getItalic()` / `setItalic()` - Gets or sets whether font is italic
- `getSize()` / `setSize()` - Gets or sets font size
- `getName()` / `setName()` - Gets or sets font name (family)
- `getUnderline()` / `setUnderline()` - Gets or sets underline style
- `getOverline()` / `setOverline()` - Gets or sets overline style
- `getStrikeout()` / `setStrikeout()` - Gets or sets strikeout style
- `getIsEmpty()` - Returns whether the font is empty
- `reset()` - Clears the font
- `mergeWith()` - Merges font with another
- `copyFrom()` - Copies font fields from another
- `toString()` - Returns CSS style string representation
- `addAttributesToRender()` - Adds CSS attributes to renderer
