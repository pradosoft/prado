**Namespace:** `Prado\Web\UI\WebControls`

## Overview

TWizardStepType defines the enumerable type for the possible types of wizard steps.

## Key Properties/Methods

- `Auto` - Type automatically determined by position
- `Complete` - Final summary step (no interaction)
- `Start` - First step
- `Step` - Middle steps
- `Finish` - Last step before Complete
