**Namespace:** `Prado\Web\UI\WebControls`

## Overview
TExpression evaluates a PHP expression and renders the result during the rendering stage. The expression is evaluated in the context of the TExpression object itself.

## Key Properties/Methods

- `getExpression()` / `setExpression()` - Gets or sets the PHP expression to evaluate
- `render()` - Renders the evaluation result of the expression
