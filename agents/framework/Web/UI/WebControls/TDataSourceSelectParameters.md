## Overview

TDataSourceSelectParameters encapsulates parameters for data source selection operations, including paging information.

## Key Properties/Methods

- `StartRowIndex` - Zero-based starting row index
- `MaximumRows` - Maximum rows to return
- `RetrieveTotalRowCount` - Whether to retrieve total count
- `TotalRowCount` - Total number of rows available
