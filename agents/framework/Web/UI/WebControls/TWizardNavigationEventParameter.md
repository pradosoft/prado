## Overview

TWizardNavigationEventParameter represents the parameter for TWizard navigation events. It contains current and next step indices, and allows cancellation of navigation.

## Key Properties/Methods

- `getCurrentStepIndex()` - Zero-based index of currently active step
- `getNextStepIndex()` / `setNextStepIndex($index)` - Zero-based index of next step
- `getCancelNavigation()` / `setCancelNavigation($value)` - Whether to cancel navigation
