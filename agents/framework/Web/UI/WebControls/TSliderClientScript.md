**Namespace:** `Prado\Web\UI\WebControls`

## Overview

TSliderClientScript manages client-side slider events. It extends TClientSideOptions and provides OnChange and OnSlide event handlers.

## Key Properties/Methods

- `getOnChange()` / `setOnChange($javascript)` - JavaScript executed when slider value changes
- `getOnSlide()` / `setOnSlide($javascript)` - JavaScript executed when slider moves
