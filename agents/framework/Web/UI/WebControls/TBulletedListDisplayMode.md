## Overview

TBulletedListDisplayMode defines how items are displayed in a TBulletedList.

## Constants

- `Text` - Items displayed as plain texts
- `HyperLink` - Items displayed as hyperlinks
- `LinkButton` - Items displayed as link buttons that cause postbacks
