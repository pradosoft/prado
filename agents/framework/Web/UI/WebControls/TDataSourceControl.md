## Overview

TDataSourceControl is an abstract base class for data source controls. Implements IDataSource interface.

## Key Properties/Methods

- `getView()` - Returns a data view by name
- `getViewNames()` - Returns available view names
- `onDataSourceChanged()` - Raises OnDataSourceChanged event
- `getVisible()` - Always returns false (not rendered)
