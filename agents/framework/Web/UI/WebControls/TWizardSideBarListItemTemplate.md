## Overview
TWizardSideBarListItemTemplate is the default template for each sidebar item in a TWizard. It creates a TLinkButton for navigation between wizard steps.

## Key Properties/Methods

- `instantiateIn($parent)` - Creates a TLinkButton with ID_SIDEBAR_BUTTON
