## Overview
TBulletStyle defines enumerable values for bullet styles in TBulletedList controls.

## Constants

- `NotSet` - Style not specified
- `None` - No bullets
- `Numbered` - Numeric ordering (1, 2, 3...)
- `LowerAlpha` - Lowercase letters (a, b, c...)
- `UpperAlpha` - Uppercase letters (A, B, C...)
- `LowerRoman` - Lowercase roman numerals (i, ii, iii...)
- `UpperRoman` - Uppercase roman numerals (I, II, III...)
- `Disc` - Filled circle
- `Circle` - Empty circle
- `Square` - Square marker
- `CustomImage` - Custom image for bullet
