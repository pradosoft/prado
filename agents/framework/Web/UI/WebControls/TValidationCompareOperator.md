## Overview
TValidationCompareOperator defines the enumerable type for comparison operations used by TCompareValidator. It specifies how two values should be compared.

## Values

- `Equal` - Values must be equal
- `NotEqual` - Values must not be equal
- `GreaterThan` - First value greater than second
- `GreaterThanEqual` - First value greater than or equal to second
- `LessThan` - First value less than second
- `LessThanEqual` - First value less than or equal to second
