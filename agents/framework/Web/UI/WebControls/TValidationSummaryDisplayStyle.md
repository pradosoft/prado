**Namespace:** `Prado\Web\UI\WebControls`

## Overview
TValidationSummaryDisplayStyle defines the enumerable type for how a TValidationSummary displays error messages.

## Values

- `None` - Error messages not displayed
- `Dynamic` - Messages dynamically shown/hidden as validators fail
- `Fixed` - Messages physically occupy page layout (may be hidden via CSS)
