## Overview
TTableRow displays a table row. Cells can be accessed via the Cells property. Supports horizontal and vertical alignment, and table section (Header, Body, Footer).

## Key Properties/Methods

- `getCells()` - Collection of TTableCell controls
- `getHorizontalAlign()` / `setHorizontalAlign($value)` - Horizontal alignment
- `getVerticalAlign()` / `setVerticalAlign($value)` - Vertical alignment
- `getTableSection()` / `setTableSection($value)` - Table section (Header, Body, Footer)
