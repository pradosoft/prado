# Web/UI/WebControls/assets/SUMMARY.md

Static assets (CSS, fonts, images) and server-side CAPTCHA image generator published to web-accessible directory by `TAssetManager`.

## Files

- **`accordion.css`** — Default styles for `TAccordion` control.

- **`tabpanel.css`** — Default styles for `TTabPanel` control.

- **`keyboard.css`** — Default styles for `TKeyboard` control.

- **`TSlider/TSlider.css`** — Default styles for `TSlider` control.

- **`TSlider/TSliderHandleHorizontal.png`** — Horizontal slider handle graphic.

- **`TSlider/TSliderHandleVertical.png`** — Vertical slider handle graphic.

- **`captcha.php`** — CAPTCHA image generator with themes: `opaque_bubble`, `noisy`, `grid`, `scribble`, `morph`, `shadowed`; requires `verase.ttf`.

- **`verase.ttf`** — TrueType font for CAPTCHA text rendering.
