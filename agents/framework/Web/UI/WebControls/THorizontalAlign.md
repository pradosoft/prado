## Overview

THorizontalAlign defines the enumerable type for possible horizontal alignments in CSS styles. Note: deprecated, use CSS text-align property instead.

## Key Properties/Methods

- `NotSet` - Alignment not specified
- `Left` - Left aligned
- `Right` - Right aligned
- `Center` - Center aligned
- `Justify` - Begin and end are justified
