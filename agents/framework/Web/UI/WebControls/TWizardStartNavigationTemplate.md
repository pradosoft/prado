## Overview

TWizardStartNavigationTemplate is the default navigation template for the first wizard step. It creates Next and Cancel buttons using the wizard's start navigation style settings.

## Key Properties/Methods

- `instantiateIn($parent)` - Creates Next and Cancel buttons
