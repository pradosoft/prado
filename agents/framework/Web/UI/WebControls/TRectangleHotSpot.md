## Overview
TRectangleHotSpot defines a rectangular clickable region in an ImageMap control. The rectangle is defined by left, top, right, and bottom coordinates. This hotspot renders with shape "rect" and coordinates in "left,top,right,bottom" format.

## Key Properties/Methods

- `getLeft()` / `setLeft(int)` - X coordinate of left side
- `getTop()` / `setTop(int)` - Y coordinate of top side
- `getRight()` / `setRight(int)` - X coordinate of right side
- `getBottom()` / `setBottom(int)` - Y coordinate of bottom side
- `getShape()` - Returns 'rect'
- `getCoordinates()` - Returns coordinate string for rendering
