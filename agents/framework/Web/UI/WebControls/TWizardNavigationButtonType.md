# TWizardNavigationButtonType

**Namespace:** `Prado\Web\UI\WebControls`

## Overview

TWizardNavigationButtonType defines the enumerable type for the type of buttons used in TWizard navigation. Each button can be rendered as a standard button, image button, or hyperlink.

## Values

- `Button` - Standard click button
- `Image` - Image button with custom image
- `Link` - Hyperlink button
