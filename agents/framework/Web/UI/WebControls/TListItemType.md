**Namespace:** `Prado\Web\UI\WebControls`

## Overview

TListItemType defines the enumerable type for possible types that databound list items can take in data grid controls.

## Key Properties/Methods

- `Header` - Header item
- `Footer` - Footer item
- `Item` - Content item (neither header nor footer)
- `Separator` - Separator between items
- `AlternatingItem` - Alternating content item
- `EditItem` - Content item in edit mode
- `SelectedItem` - Selected content item
- `Pager` - Pager
