## Overview
TTableHeaderRow displays a table header row. It extends TTableRow and always returns 'Header' for getTableSection(), making it read-only.

## Key Properties/Methods

- `getTableSection()` - Always returns 'Header'
- `setTableSection($value)` - Throws exception (read-only)
