## Overview

TItemDataRenderer is a convenient base class for item data renderers in repeater/list controls. Implements IItemDataRenderer interface.

## Key Properties/Methods

- `ItemIndex` - Zero-based index in item collection
- `ItemType` - Type of item (Item, AlternatingItem, etc.)
- `Data` - Data associated with this renderer
