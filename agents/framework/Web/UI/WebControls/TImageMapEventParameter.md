**Namespace:** `Prado\Web\UI\WebControls`

## Overview

TImageMapEventParameter represents a postback event parameter when a hotspot is clicked in an ImageMap control. It provides access to the PostBackValue associated with the clicked hotspot.

## Key Properties/Methods

- `getPostBackValue()` - Returns the post back value associated with the clicked hotspot
