## Overview

TReadOnlyDataSourceView represents a read-only view of data from a TReadOnlyDataSource. It extends TDataSourceView and implements the select method to return data.

## Key Properties/Methods

- `__construct(IDataSource $owner, $viewName, $dataSource)` - Constructor
- `select($parameters)` - Returns the data source
