**Namespace:** `Prado\Web\UI\WebControls`

## Overview
TLinkButton creates a hyperlink-style button that posts back to the page. Can be a submit button or command button with parameters.

## Key Properties/Methods

- `Text` - Button caption
- `CommandName` - Command name for OnCommand event
- `CommandParameter` - Parameter for command
- `CausesValidation` - Whether button triggers validation
- `ValidationGroup` - Validation group to trigger
- `EnableClientScript` - Whether to render JavaScript
- `onClick()` - Click event
- `onCommand()` - Command event (bubbles)
- `raisePostBackEvent()` - Handles postback
