## Overview

IDataSource defines the interface that data source controls must implement to provide data views.

## Key Properties/Methods

- `getView($viewName)` - Returns a data view by name
- `getViewNames()` - Returns array of available view names
- `onDataSourceChanged($param)` - Called when data source changes
