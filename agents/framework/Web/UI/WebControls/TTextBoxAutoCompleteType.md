## Overview
TTextBoxAutoCompleteType defines the possible AutoComplete types supported by TTextBox controls. AutoComplete provides suggestions as users type in text boxes.

## Values

- `Disabled` - AutoComplete disabled
- `Enabled` - AutoComplete enabled
- `None` - No specific type
