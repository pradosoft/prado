**Namespace:** `Prado\Web\UI\WebControls`

## Overview
TEditCommandColumn contains Edit/Update/Cancel command buttons for editing data items in a DataGrid row.

## Key Properties/Methods

- `ButtonType` - Type of buttons (LinkButton, PushButton, ImageButton)
- `EditText` / `UpdateText` / `CancelText` - Button captions
- `EditImageUrl` / `UpdateImageUrl` / `CancelImageUrl` - Image URLs for image buttons
- `CausesValidation` - Whether Update button triggers validation
- `ValidationGroup` - Validation group for Update button
- `initializeCell()` - Creates appropriate buttons based on row state
