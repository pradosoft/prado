**Namespace:** `Prado\Web\UI\WebControls`

## Overview
TCircleHotSpot defines a circular hot spot region in a TImageMap control.

## Key Properties/Methods

- `X` / `Y` - Center coordinates of the circle
- `Radius` - Radius of the circular region
- `getShape()` - Returns 'circle'
- `getCoordinates()` - Returns 'x,y,radius' format
