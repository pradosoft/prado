## Overview

TEmailAddressValidator validates whether a value is a valid email address. Can optionally check MX records.

## Key Properties/Methods

- `CheckMXRecord` - Whether to check DNS MX records (default false)
- `EMAIL_REGEXP` - Regular expression for email validation
